    • 方法一、用Google Place API 從 Google Map上取得營業時間

      首先要從 Google Map Place API 的 Place Search 服務取得藥局的place_id ，要做到這件事，我們需要從API Request 輸入6個參數，分別是:

      • key Google Map Place API key
      • location 想找到的目的地的 緯度/經度
      • radius 搜索半徑 (公尺)；除了指定location外，還要指定搜索半徑

      可選參數

      • keyword 一段用來搜尋的文字，像是OOO藥局
      • type Google Map 上的標的會做基本的分類，像是藥局 (pharmacy)、醫院 (hospital) ...等，詳細的分類可以從這裡找到
      • language 返回的搜索結果語言，詳細的語言列表可以在這裡找到

      以離學校最近的藥局 博爾康生活藥局 來當作範例

      radius100單位為公尺

    • 方法二、一樣從政府衣料開放平台的 全民健康保險特約院所固定服務時段 | 政府資料開放平臺 取得資料

      直接透過 R join 口罩剩餘數量的 data frame 即可。

  1. 用GPS定位當前位置與附近的幾家藥局做時間預估

    ◾透過瀏覽器要求GPS權限取得使用者位置後，在拿來與資料庫中的藥局位置比較。

  1. 提供使用者能透過：
    1. 地址 (文字搜尋)
    1. 地址 (下拉式選單)
    1. GPS定位

    搜尋剩餘口罩數量

    ◾此項目能夠提供人們以位置來查詢口罩剩餘數量，可透過三種方式來找尋地址，下拉式選單選取地址，將地址分段選取，如縣市、鄉鎮市區、村里，來漸進式選取位置。文字搜尋，直接輸入明確地址進行查詢，以及GPS定位系統，定位所在位置，顯示四周藥局口罩剩餘數量，藉由三種方式以便民眾查詢剩餘口罩數量。
